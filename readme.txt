== Description ==

Easy Maintenance Mode Includes:

* Enable or Disable Maintenance Mode under Settings->Easy Maintenance Mode
* Ability to customize a title and description to the maintenance mode screen
* Ability to still manage the administrative panel while in maintenance mode

== Installation ==

Upload the Easy Maintenance Mode plugin to your blog, then activate it.
